${\huge{Linear}{~}{Regression}{~}{Modeling}{~}{-}{~}{Electronic}{~}{Switch}{~}{Failure}}$

One way of looking at linear regression models is to think of them as modeling ${RESPONSE} = {STIMULUS} + {ERROR}$. In other words, the response (or dependent variable or target value or predicted value) ${(Y)}$ is related linearly to a stimulus (or independent variable or predictor) ${(X)}$ plus some error ${(e)}$. To see how it fits in with some related models, please see the summary below.

In ANOVA the stimulus is fixed¶

${\large{Y} = {X} + {error}}$

In Regression the stimulus is random¶

${\large{Y} = {\beta}{X}+{error}}$

In Factor Analysis the stimulus is latent¶

${\large{Y} = {\lambda}{X} + {error}}$

-------------------------------------------------------------------------------¶

${\huge{Simple}{~}{Linear}{~}{Regression}{~}{Model}}$

The simple linear regression model is based on a set of ordered pairs (i.e. sample data) where the i-th ordered pair are related to each other as follows.

$${\Large {y_i = {a}{~}+{~}{bx_i}{~}+{~}{e_i}}} $$

We also know that the i-th predicted response $({\hat{y_i}})$ is defined as follows.

$${\Large{\hat{y_i}}={a}{~}+{~}{bx_i}}$$

Thereby allowing for the calculation of the error (also known as the residual).

$${\Large{y_i} - {\hat{y_i}}={e_i}}$$

Since the error can be positive or negative, the error is squared. Summing all of the squared errors gives us the Sum of Squared Errors - ${{SSE}}$.

$$\Large{SSE =}\sum\limits_{i=1}^n (y_i-{\hat{y_i})^2}$$

Develpoment of the regression line ${a}{~}+{~}{bx_i}$ is based on minimizing the ${SSE}$.

Once the simple linear regression model has been developed it is necessary to check if the model is appropriate. This is accomplished by checking the assumptions associated with linear regression. The assumptions are that the errors, $e_i$ are independent, normally distributed, random variables with a mean of zero and a constant variance, $\sigma_i^2$. Residual analysis is used to determine the fitness of applying linear regression.

Example - Model failure of a new electronic switch¶

In this example a company has just developed a new electronic switch. The company is concerned about warranty claims and wants to be able to provide customers with the ability to predict the failure rate of the switch.

Using Python and data provided by the engineering group, attempt to develop a Linear Regression model that the company can use to avoid warranty claims.

First, it is known that any data point that falls directly on the regression line will have a residual of 0; as a result, the residual = 0 line directly corresponds to the regression line. Second, the residuals vs. fits plot appears to be well-behaved and confirms the following assumptions when applying a linear regression model:

  • The residuals appear to be randomly distributed around the residual = 0 line. This supports the assumption that the relationship is linear.
  • The residuals form a horizontal band around the residual = 0 line. This supports the assumption that the variances of the errors are approximately equal.
  • There are no residuals that stand out from the rest. This supports the assumption of no outliers.

Next, please see the coefficient of determination below.

In [7]:
print('Coefficient of Determination: %.3f' % r2_score(sm_lst_failure_rate_y, predictions))

Coefficient of Determination: 0.937

The coeficient of determination $(r^2)$ represents the portion of the variation of the dependent variable (the failure rate/1000 cycles) that is explained by the independent variable (the number of cycles). If all of the variation in the dependent variable had been explained by the predictor then $(r^2)$ would have been equal to one (1). This, along with the residuals analysis, supports the application of the regression model.

Finally, the creation of a residuals vs. predictor plot will assist in determining the value of the time data that was submitted by the engineering team. In this type of plot the residuals (plotted on the y-axis) already include failure rate adjusted for number of cycles; the new predictor, time, is plotted on the x-axis. If the residuals vs. predictor plot displays a non-random pattern, it indicates that the predictor explains some of the remaining variability in the residuals. This would support the addition of another independent variable to the model, leading to the development of a multiple regression model.

The one caveat is that the model is only applicable for the range of cycle numbers between 12000 and 24000 inclusive.
